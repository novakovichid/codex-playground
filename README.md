# Codex Playground

Этот репозиторий является мультипроектным и собирает независимые приложения в отдельных подкаталогах. Детали настройки и запуска каждого проекта описаны в их собственных README.

## Проекты
- **SpeechTrap** — электронная версия настольной игры Taboo с фронтендом на React и backend на FastAPI. Исходный код и инструкции находятся в каталоге [`speechtrap/`](speechtrap/README.md).

При добавлении новых проектов размещайте их в отдельных папках рядом с `speechtrap/` и дополняйте этот список.
