# Codex Playground

Этот репозиторий является мультипроектным и собирает независимые приложения в отдельных подкаталогах. Детали настройки и запуска каждого проекта описаны в их собственных README.

## Проекты
- **SpeechTrap** — электронная версия настольной игры Taboo с фронтендом на React и backend на FastAPI. Быстрый вход во фронтенд: [`speechtrap/frontend/index.html`](speechtrap/frontend/index.html). Инструкции по сборке и бэкенду лежат в [`speechtrap/README.md`](speechtrap/README.md).
- **SerboLingo** — статичное приложение в духе Duolingo для изучения сербского языка. Готово к публикации, главный вход: [`serbian-duo/index.html`](serbian-duo/index.html).
- **Mappu (GitHub Pages)** — статический MVP интерактивных карт без серверной части, главный вход: [`maps-gh-pages/index.html`](maps-gh-pages/index.html).
- **Mappu (Django)** — полнофункциональное приложение интерактивных карт; в репозитории доступна только документация и схемы в [`README - maps.md`](README%20-%20maps.md).

Полный ссылочник подпроектов доступен в файле [`PROJECTS_INDEX.md`](PROJECTS_INDEX.md).

При добавлении новых проектов размещайте их в отдельных папках рядом с существующими и обновляйте оба списка.
