# Ссылочник подпроектов Codex Playground

Этот репозиторий хранит несколько независимых демо-приложений. Ниже собраны прямые ссылки на фронтенд каждой доступной части.

- [SpeechTrap — фронтенд](speechtrap/frontend/index.html) — Vite/React интерфейс электронной версии настольной игры Taboo. Бэкенд и инструкции по запуску находятся в [`speechtrap/README.md`](speechtrap/README.md).
- [SerboLingo](serbian-duo/index.html) — статичное приложение в духе Duolingo для изучения сербского, готовое к публикации на GitHub Pages.
- [Mappu (GitHub Pages)](maps-gh-pages/index.html) — статический MVP интерактивных карт, работающий в браузере без серверной части.
- [Mappu (Django)](README%20-%20maps.md) — полнофункциональное приложение интерактивных карт; в репозитории доступна только документация и схемы.

При добавлении новых подпроектов обновляйте этот ссылочник и корневой README.
