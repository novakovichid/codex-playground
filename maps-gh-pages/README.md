# Mappu для GitHub Pages

Статический MVP интерактивных карт, собранный по README проекта «Mappu». Все работает без серверной части и сохраняется в LocalStorage, поэтому его можно публиковать в подкаталоге репозитория и подключать к GitHub Pages.

## Возможности
- Интерактивные карты с узлами и связями.
- Поиск по названию и хэштегам, фильтр «Только публичные».
- Профиль автора (имя, email, статус, аватар) с хранением в LocalStorage.
- Публичные/приватные карты, дублирование и удаление.
- Публикация карты в виде share-ссылки (данные кодируются в hash), автоматический импорт по ссылке.
- Импорт/экспорт JSON, копирование в буфер обмена.
- Редактор узлов с перетаскиванием, авторасстановка по кругу и добавление связей.

## Запуск
Папка уже готова для GitHub Pages: достаточно указать подкаталог `maps-gh-pages` в настройках Pages или открыть файл `index.html` локально.

1. Откройте `maps-gh-pages/index.html` в браузере (двойной клик или через локальный сервер).
2. Настройте профиль, создайте или выберите карту, добавляйте узлы/связи.
3. Кнопка «Опубликовать ссылку» создаст hash-ссылку, которую можно отправить — получатель импортирует карту автоматически.

Никаких внешних зависимостей нет. Все стили и логика в `styles.css` и `app.js` внутри каталога.
