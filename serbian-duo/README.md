# SerboLingo

Мини-приложение в стиле Duolingo для изучения сербского носителем русского языка. Полностью статично, готово к GitHub Pages — просто откройте `index.html` или задеплойте каталог как есть.

## Возможности
- Два режима: выбор правильного перевода и ручной ввод ответа.
- Обширный словарь по темам (приветствия, еда, путешествия, эмоции, числа и др.).
- Прогресс, серии, жизни и уровень владения словами сохраняются в Local Storage.
- Адаптивный интерфейс под iPhone 15 Pro Max и экраны 16:9, с учетом safe-area.
- Мгновенные подсказки (транслитерация, примечания) прямо в заданиях.

## Как использовать
1. Откройте `index.html` локально или разместите каталог в GitHub Pages.
2. Выберите режим и выполняйте 10 вопросов за сессию.
3. Собирайте XP, следите за серией и отмечайте категории словаря — данные сохраняются в браузере.

## Данные
Словарь встроен в `app.js` и охватывает бытовые темы, путешествия, чувства и числа. По клику на категорию видно переводы, транслитерацию и уровень освоения каждого слова.
